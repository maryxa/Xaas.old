Adds file to vector<string> XmlFiles
All files in the vector will be added in the generated Dom Object.
The search order for the file:
1) Common shared Directory SETTINGS["SharedDir"]+"/xml/"
2) Application Xml Directory SETTINGS["XmlDir"]
The 2 parameters: SharedDir and XmlDir are configurable in the configuration file
