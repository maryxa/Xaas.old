Adds file to vector<string> XmlFiles
All files in the vector will be added in the generated Dom Object.
The search order for the file:
1) Application Xml Directory SETTINGS["XmlDir"]
2) Common shared Directory SETTINGS["SharedDir"]+"/xml/"
The 2 parameters: SharedDir and XmlDir are configurable in the configuration file

Builds the XML for the Back End Call
The general structure is :

<WsData>
<login>
<token>[TOKEN]</token>
</login>
<operation>
<object>[OBJECT]</object>
<event>[EVENT]</event>
</operation>
<params>
<p>
<n></n>
<v></v>
</p>
</params>
</WsData>
